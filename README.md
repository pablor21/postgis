# POSGIS PostgreSQL Image

This repository contains a Docker image that extends the official PostgreSQL image by adding PostGIS extensions. It is designed to provide a ready-to-use PostgreSQL database with spatial capabilities.

## Features
- Based on the official PostgreSQL Docker image.
- Includes PostGIS extensions for spatial data support.
- Supports multiple PostgreSQL and PostGIS versions.
- Easy to use and deploy in various environments.


## Supported Architectures

- amd64
- arm64v8
